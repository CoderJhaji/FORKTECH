# RasSetu

An AI-powered recipe regeneration app that adapts any dish to your dietary constraints and allergies while preserving authentic flavor.

## Development

Requirements:

- Node.js and npm

Install dependencies:

```sh
npm install
```

Run the development server:

```sh
npm run dev
```

Run tests:

```sh
npm test
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
